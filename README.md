# Sistema de Arquivos em Python

Este projeto é uma implementação de um sistema de arquivos simples em Python. Ele usa um arquivo de texto como armazenamento e fornece uma interface de linha de comando para interagir com o sistema de arquivos.

## Funcionalidades

O sistema de arquivos suporta os seguintes comandos:

- `mkdir`: Cria um novo diretório.
- `rmdir`: Remove um diretório existente.
- `ls`: Lista os arquivos e diretórios no diretório atual.
- `cd`: Muda o diretório atual.
- `mv`: Move ou renomeia arquivos e diretórios.
- `ln`: Cria um link simbólico para um arquivo ou diretório.
- `touch`: Cria um novo arquivo.
- `echo`: Escreve texto em um arquivo.
- `rm`: Remove um arquivo.
- `cat`: Mostra o conteúdo de um arquivo.
- `cp`: Copia um arquivo.
- `clear`: Limpa a tela do terminal.
- `clearALL`: Limpa todo o sistema de arquivos.
- `exit`: Sai do sistema de arquivos.

## Como usar

Para usar o sistema de arquivos, execute o arquivo `main.py`. Você será apresentado a um prompt de comando onde pode digitar os comandos listados acima.

## Requisitos

Este projeto requer Python 3.6 ou superior.
